# Button Components

## Quick Import

```tsx
// Import individual components
import { AddToQuoteButton } from '@/components/buttons/ActionButtons';

// Or import from index
import { AddToQuoteButton, ProductActions } from '@/components/buttons';
```

## Components

### AddToQuoteButton
Adds products to quotes with loading and success states.

### AddToBulkOrderButton
Adds products to bulk orders. Supports checkbox and button modes.

### ContactMerchantButton
Initiates contact with merchants.

### ProductActions
Combined action group for product cards.
