# Modal Components

## Quick Import

```tsx
// Import individual components
import { Modal } from '@/components/modals/Modal';

// Or import from index
import { Modal, ConfirmModal } from '@/components/modals';
```

## Components

### Modal
Full-featured modal with size variants and animations.

### ConfirmModal
Specialized modal for confirmation dialogs.

### ModalHeader, ModalBody, ModalFooter
Building blocks for custom modal layouts.
